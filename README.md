# CuteCats

CuteCats — это веб-приложение для работы с котятами. Вы можете создавать, изменять, удалять и оценивать котят.

## Технологии
- **Django**
- **Django Rest FrameWork** и дополнительные библиотеки
- **Nginx**
- **Docker**, **Docker-compose**

## Запуск приложения
0. Убедитесь, что **Docker** и **Docker Compose** установлены и запущены на вашей системе.
1. Перейдите в корень проекта (папка с файлом `Dockerfile`).
2. Откройте консоль в этой папке.
3. Выполните команду:
   ```bash
   docker-compose up --build -d
   ```
4. Зайдите на [http://127.0.0.1/swagger/](http://127.0.0.1/swagger/). Там будет описан API, с помощью которого можно взаимодействовать с проектом.

## Документация API

### Обзор
Этот API предоставляет конечные точки для управления породами кошек и самими кошками. Он поддерживает регистрацию пользователей и аутентификацию с использованием токенов JWT.

### Базовый URL
```
http://localhost/api/v1
```

### Аутентификация
Этот API использует базовую аутентификацию. Включите ваше имя пользователя и пароль в заголовок запроса.

### Конечные точки
1. **Получить породы**
   - **GET** `/breed`
   - Описание: Получить список всех пород кошек.
   - Ответ: Массив пород.

2. **Список кошек**
   - **GET** `/cat/`
   - Описание: Получить список кошек. Можно фильтровать по породе или пользователю.
   - Параметры:
     - `breed` (необязательно): Фильтрация по имени породы.
     - `user` (необязательно): Фильтрация по ID пользователя.
   - Ответ: Массив кошек.

3. **Создать кошку**
   - **POST** `/cat/`
   - Описание: Создать новую кошку.
   - Тело запроса:
     ```json
     {
       "color": "string",
       "age": "integer",
       "description": "string",
       "breed": "integer"
     }
     ```
   - Ответ: Объект созданной кошки.

4. **Получить кошку по ID**
   - **GET** `/cat/{id}/`
   - Описание: Получить конкретную кошку по её ID.
   - Параметры:
     - `id` (обязательно): Уникальный идентификатор кошки.
   - Ответ: Объект кошки.

5. **Обновить кошку**
   - **PUT** `/cat/{id}/`
   - Описание: Обновить существующую кошку.
   - Параметры:
     - `id` (обязательно): Уникальный идентификатор кошки.
   - Тело запроса: Структура аналогична созданию кошки.
   - Ответ: Объект обновлённой кошки.

6. **Частичное обновление кошки**
   - **PATCH** `/cat/{id}/`
   - Описание: Частично обновить существующую кошку.
   - Параметры:
     - `id` (обязательно): Уникальный идентификатор кошки.
   - Тело запроса: Частичный объект кошки.
   - Ответ: Объект обновлённой кошки.

7. **Удалить кошку**
   - **DELETE** `/cat/{id}/`
   - Описание: Удалить конкретную кошку по её ID.
   - Параметры:
     - `id` (обязательно): Уникальный идентификатор кошки.
   - Ответ: Нет содержимого.

8. **Оценить кошку**
   - **POST** `/cat/{id}/rate/`
   - Описание: Оценить конкретную кошку.
   - Параметры:
     - `id` (обязательно): Уникальный идентификатор кошки.
   - Тело запроса: Структура аналогична созданию кошки.
   - Ответ: Объект оценённой кошки.

9. **Получить рейтинг кошки**
   - **GET** `/cat/{id}/rating/`
   - Описание: Получить рейтинг конкретной кошки.
   - Параметры:
     - `id` (обязательно): Уникальный идентификатор кошки.
   - Ответ: Объект кошки с деталями рейтинга.

10. **Регистрация пользователя**
    - **POST** `/register`
    - Описание: Зарегистрировать нового пользователя.
    - Ответ: Нет содержимого.

11. **Получить токен**
    - **POST** `/token`
    - Описание: Получить токен JWT, используя учётные данные пользователя.
    - Тело запроса:
      ```json
      {
        "username": "string",
        "password": "string"
      }
      ```
    - Ответ: Токены доступа и обновления.

12. **Обновить токен**
    - **POST** `/token/refresh`
    - Описание: Обновить токен JWT, используя токен обновления.
    - Тело запроса:
      ```json
      {
        "refresh": "string"
      }
      ```
    - Ответ: Новый токен доступа.

## Модели данных

### Порода
- **ID**: integer (только для чтения)
- **Имя**: string (обязательно, maxLength: 100)

### Кошка
- **ID**: integer (только для чтения)
- **Цвет**: string (обязательно, maxLength: 100)
- **Возраст**: integer (обязательно)
- **Описание**: string (обязательно, maxLength: 1000)
- **Порода**: integer (обязательно)
- **Пользователь**: integer (только для чтения)
- **URL списка рейтингов**: string (только для чтения)
- **Количество рейтингов**: string (только для чтения)
- **Средний рейтинг**: string (только для чтения)

### TokenObtainPair
- **Имя пользователя**: string (обязательно)
- **Пароль**: string (обязательно)

### TokenRefresh
- **Refresh**: string (обязательно)
- **Access**: string (только для чтения)
